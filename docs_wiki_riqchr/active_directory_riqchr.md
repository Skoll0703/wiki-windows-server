# Active Directory Domain Services

## Objetivo

Instalar y configurar Active Directory para convertir el servidor en un controlador de dominio.

## Pasos principales

1. Instalar el rol AD DS en Windows Server.
2. Promover el servidor a controlador de dominio.
3. Crear el dominio inacap.local.
4. Definir usuarios, equipos y unidades organizativas.

## Resultado esperado

El servidor se convierte en el centro de autenticación y administración del laboratorio.
