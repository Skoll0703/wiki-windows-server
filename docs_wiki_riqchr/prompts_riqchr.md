# Prompts para IA

## Uso recomendado

Estos prompts pueden utilizarse para pedir apoyo a una inteligencia artificial en tareas de administración del laboratorio Windows Server.

## Ejemplos de prompts

- Diseñar una política de seguridad para el dominio.
- Preparar un equipo Windows para unirse a inacap.local.
- Diagnosticar problemas de DNS y DHCP.

## Prompt base

> Actúa como administrador de Windows Server y ayúdame a resolver esta tarea de forma segura y práctica.
