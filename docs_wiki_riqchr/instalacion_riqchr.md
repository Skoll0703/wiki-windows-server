# Instalación y configuración básica

## Objetivo

Configurar Windows Server como controlador del laboratorio para que pueda actuar como base del dominio.

## Configuración realizada

- Nombre del equipo: SRV-DC01
- IP fija: 192.168.10.10
- Firewall configurado para servicios básicos
- Red interna preparada para comunicación entre servidor y cliente
