# Políticas de grupo (GPO)

## Objetivo

Aplicar configuraciones centralizadas a equipos y usuarios del dominio mediante políticas de grupo.

## Ejemplos de uso

- Aplicar restricciones de seguridad
- Desactivar opciones no deseadas del sistema
- Configurar accesos y permisos por grupos
- Administrar usuarios de manera homogénea

## Valor de las GPO

Las políticas de grupo ayudan a mantener el entorno más seguro, consistente y fácil de administrar.
