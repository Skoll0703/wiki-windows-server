# Inicio

## Laboratorio Windows Server

Esta wiki documenta la implementación de un laboratorio con Windows Server, incluyendo la instalación del sistema, la configuración de Active Directory, la integración de un cliente al dominio, la habilitación de servicios de red y la administración con políticas de grupo.

## Topología del laboratorio

- Servidor: SRV-DC01
- Cliente: PC01
- Dominio: inacap.local
- Red interna: redlab

## Objetivo

Mostrar un flujo completo de puesta en marcha de un entorno Windows Server, desde la instalación hasta la administración centralizada de usuarios y equipos.

## Herramientas utilizadas

- VirtualBox
- Windows Server
- Windows Client
- DNS
- DHCP
- GPO
