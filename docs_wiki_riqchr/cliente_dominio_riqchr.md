# Integración del cliente al dominio

## Objetivo

Unir un equipo cliente al dominio para permitir la autenticación centralizada y el acceso a recursos compartidos.

## Acciones en el cliente

- Asignar el DNS del servidor controlador
- Cambiar el nombre del equipo
- Unirse al dominio inacap.local
- Iniciar sesión con una cuenta de dominio

## Validación

Una vez unido al dominio, el equipo queda listo para usar los recursos y permisos definidos por Active Directory.
