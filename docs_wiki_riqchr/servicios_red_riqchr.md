# Servicios de red: DNS y DHCP

## Objetivo

Configurar servicios de red que permitan la resolución de nombres y la asignación automática de direcciones IP.

## Funciones principales

- DNS para resolver nombres del dominio
- DHCP para asignar IP de forma automática
- Alcances y reservas configurados dentro del rango de red

## Importancia

Estos servicios hacen que la red del laboratorio se comporte de forma más realista y que los equipos se conecten de manera automática y ordenada.
